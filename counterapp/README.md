# Create Counter App (using React)

## Requirements:

- Your app should be able to:

  - Increase by click
  - Decrease by click
  - Reset it to 0 by click
  - Make decrease button disable if number <= 0

- Add three inputs

  - For maximal value that your counter can have
  - For minimal value
  - Step

- Save all info in local storage
